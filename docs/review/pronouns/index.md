# Pronouns


## Reflexive pronoun

Refers to ("reflects") the same person or thing that is the *subject* of the clause.

| Case | 	Sing./Pl. |
| --- | --- |
| Nominative | 	— |
| Genitive | 	suī |
| Dative | 	sibi |
| Accusative | 	sē |
| Ablative | 	sē |
